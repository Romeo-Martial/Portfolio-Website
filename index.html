About Me

I’m a France-based Frontend Web Developer focused on building clean, responsive, and user-centered web applications.

I recently designed and deployed a React CV Builder application that allows users to create and edit structured resumes through dynamic forms with live preview. Through this project, I strengthened my skills in component-based architecture, state-driven UI, and building real, production-style frontend workflows.

My core technical stack includes HTML, CSS, JavaScript (ES6+), React, Bootstrap, Sass, and Python. I enjoy turning ideas into functional interfaces that are both visually clear and technically well-structured.

Coming from a mathematics and engineering background, I approach development with strong analytical thinking, attention to detail, and a structured problem-solving mindset. I value clean code, maintainability, and continuous improvement through real projects.

I’m currently open to junior frontend roles and freelance opportunities, and I enjoy collaborating with both French and international teams to build impactful web experiences.
